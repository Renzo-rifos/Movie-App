## ⚙️ Tech Stack

- React 19
- Vite
- Tailwind CSS
- TMDB API
- Appwrite

---

## ✨ Features

- 🔍 Debounced real-time movie search
- 📈 Trending movies based on search history
- 🎥 Movie cards with poster, title and rating
- ⏳ Loading spinner while fetching data
- 📱 Fully responsive design

---

## 🛠️ Run locally
```bash
git clone https://github.com/Renzo-rifos/Movie-App.git
cd Movie-App
npm install
npm run dev
```

### Environment variables

Create a `.env` file in the root:
```env
VITE_TMDB_API_KEY=your_tmdb_key
VITE_APPWRITE_PROJECT_ID=your_project_id
VITE_APPWRITE_DATABASE_ID=your_database_id
VITE_APPWRITE_TABLE_ID=metrics
```
